# Landing Page - The Odin Project

This project is part of [The Odin Project](https://www.theodinproject.com/) curriculum. The goal was to build a responsive landing page based on a provided design mockup.

## Project Overview
The project was developed following a set of design specifications provided in two parts:
* **Layout Design:** A full-page visual reference to guide the structure and positioning.
* **Style Guide:** A breakdown of the color palette and typography used throughout the site.
